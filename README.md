# dota2_watcher_qqbot
dota2 watcher qq bot with coolq
修改自 https://github.com/unilink233/dota2_watcher

用酷Q搞了个bot, 自动给群里推送战绩

用法: 

- 先装个酷Q的docker: https://cqhttp.cc/docs/4.15/#/Docker
- 去NoVNC登陆, 密码MAX8char
- 在服务器用nohup后台运行脚本即可: nohup python -u task.py > test.log 2>&1 &
